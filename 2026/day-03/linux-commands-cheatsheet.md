## Linux Commands Cheatsheet

### 1. Process Management
- ps aux:	Show all running processes
- ps -ef:	Detailed process list
- top:	Live view of CPU & memory usage
- htop:	Better interactive process viewer (My fav.)
- kill -9 <PID>: Force stop a process

### 2. File System Commands
- ls:	List files
- ls -la:	Detailed file list with permissions
- pwd:	Show current directory
- cd <dir>:	Change directory
- mkdir <dir>:	Create directory
- rm <file>:	Delete file
- rm -r <dir>:	Delete directory
- cat <file>: View file content
- df -h:	Disk space usage
- chmod:	Change permissions
- chown:	Change owner
- chgrp: change both owner and group

### 3. Networking
- ip a:	Show IP addresses
- ip r:	Show routing table
- ping <host>:	Check network connectivity
- curl <url>:	Test API / service response
- nslookup <domain>:	DNS check
- dig <domain>:	Detailed DNS info
- traceroute <host>:	Network path check
- curl ifconfig.me: gives the ip address of your system
